# Hotel Booking Management System

ระบบจัดการการจองห้องพักโรงแรม เขียนด้วยภาษา C

## คุณสมบัติหลัก

- **เพิ่มการจอง** - บันทึกข้อมูลการจองห้องพักพร้อมคำนวณราคาอัตโนมัติ
- **ยกเลิกการจอง** - ยกเลิกการจองที่มีอยู่
- **ชำระเงิน** - จัดการการชำระเงินค่าห้องพัก
- **ค้นหาข้อมูล** - ค้นหาและแสดงรายละเอียดการจอง
- **รายงานสรุป** - สรุปสถิติการจองและรายได้
- **บันทึกข้อมูล** - บันทึกข้อมูลลงไฟล์

## ประเภทห้องและราคา

| ประเภทห้อง | ราคาต่อคืน |
|-----------|-----------|
| Standard  | 1,200 THB |
| Deluxe    | 1,700 THB |
| Suite     | 2,200 THB |

## การใช้งาน

### คอมไพล์โปรแกรม

```bash
gcc hotel_booking.c -o hotel_booking
```

### รันโปรแกรม

**Windows:**
```bash
hotel_booking.exe
```

**Linux/Mac:**
```bash
./hotel_booking
```

## เมนูระบบ

```
1. Add Booking              - เพิ่มการจองใหม่
2. Cancel Booking           - ยกเลิกการจอง
3. Process Payment          - ชำระเงิน
4. Search and Display       - ค้นหาข้อมูลการจอง
5. View Summary Report      - ดูรายงานสรุป
6. Save Bookings            - บันทึกข้อมูล
7. Exit Program             - ออกจากโปรแกรม
```

## โครงสร้างข้อมูล

โปรแกรมบันทึกข้อมูลต่อไปนี้สำหรับแต่ละการจอง:

- **ID การจอง** - รหัสอัตโนมัติ
- **ชื่อ-นามสกุล** - ข้อมูลผู้จอง
- **ประเภทห้อง** - Standard/Deluxe/Suite
- **ราคารวม** - คำนวณจากจำนวนคืน × ราคาต่อคืน
- **วันเช็คอิน/เช็คเอาท์** - รูปแบบ dd/mm/yyyy
- **สถานะการจอง** - Active หรือ Canceled
- **สถานะการชำระเงิน** - Paid หรือ Unpaid

## การจัดเก็บข้อมูล

ข้อมูลถูกบันทึกในไฟล์ `bookings.txt` ในรูปแบบ:

```
[ID] [ชื่อ] [นามสกุล] [ประเภทห้อง] [ราคา] [เช็คอิน] [เช็คเอาท์] [isActive] [isPaid]
```

## ตัวอย่างการใช้งาน

### 1. เพิ่มการจอง
```
First Name: Somchai
Last Name: Jaidee
Room Type: Deluxe
Check-in: 01/12/2024
Check-out: 03/12/2024

→ คำนวณ: 2 คืน × 1,700 = 3,400 THB
```

### 2. ค้นหาและแสดงข้อมูล
```
First Name: Somchai
Last Name: Jaidee

→ แสดงรายละเอียดการจองทั้งหมด
```

### 3. ชำระเงิน
```
First Name: Somchai
Last Name: Jaidee

→ ยืนยันชำระเงิน 3,400 THB
```

## ข้อกำหนดระบบ

- **Compiler:** GCC หรือเทียบเท่า
- **OS:** Windows, Linux, macOS
- **Memory:** จัดเก็บได้สูงสุด 100 การจอง

## หมายเหตุ

- ระบบรองรับภาษาไทย (UTF-8)
- ข้อมูลจะโหลดอัตโนมัติเมื่อเปิดโปรแกรม
- ควรบันทึกข้อมูลก่อนออกจากโปรแกรม (เลือกเมนู 6)

## ข้อจำกัด

- จัดเก็บได้สูงสุด 100 การจอง
- การคำนวณจำนวนคืนเป็นแบบง่าย (ไม่นับปีอธิกสุรทิน)
- ค้นหาได้เฉพาะชื่อ-นามสกุลที่ตรงทั้งหมด
